关于 Vikings
==============
如果你在网上看到本站了，那很荣幸说明该站有一定流量了。建立该站的前期目标是对自己的知识体系进行梳理并总结，高尔基说过“书籍是人类进步的阶梯”，我们能享受互联网上丰富的文档资料离不开前辈的无私奉献。所以我想把这个精神传承下去，做一个有质量的知识学习站，当有人需要学习新的知识时能够在本站得到他想要的即可。如果学习后对知识有更好更深刻的理解再回馈的本站。

该站文档源码托管在 github 上，网站部署在 readthedoc，你如有也有兴趣参与本站的建设可以到 github 上克隆该仓库，修改或添加新的文档提交后请求合并即可，仓库管理员会进行审核，审核通过就可以在该站上看到你的提交。至于内容审核是为了提升文档质量，审核包括内容、用词
方式、文档风格等方面。总之是为了保证该站高质量、高要求的形象。用户不能在该站有错误的知识指引。

Vikings 瞄准优秀的技术学习、知识原理的讲解。做一个纯粹的学习站，在中国来说程序员更熟悉 CSDN，CSDN 虽然被许多人鄙视但是它也为许多初学者提供了不可磨灭的帮助。以及站长自己最开始也从上面开始接触别人写的文档，但是后面自己发现 CSDN 内容良莠不齐、对同一问题要获取答案可能需要翻阅多篇文档当不能确定其回答的准确性、用户得不到很好的阅读体验。针对以上问题，我觉得 WiKi 的就和本站相似，WiKi 专注名词的解释。而本站想专注技术的讲解，也希望通过大家的努力对于一个技术的讲解有同一的线路而不是让读者摸不到方向。

Vikings 指的是以前的北欧维京海盗，而本站就是一艘知识的海盗船，知识就是黄金，跟海盗喜欢掠夺黄金的特点一拍即合所以取名为 Vikings，欢迎新的海盗上船，我希望这艘船能先从游遍中国，最后再驶向太平洋、大西洋掠夺全世界的优秀知识，让更多的人知道 Vikings。
