关于 Vikings
==============
如果你在网上看到本站了，很荣幸说明该站有一定流量了。

建立该站是为了自己所学到的知识进行梳理总结，高尔基说过“书籍是人类进步的阶梯”，我们能享受互联网上丰富的文档资料离不开前辈的无私奉献。所以我想把这个精神传承下去，做一个有质量的知识学习站，一来可以提高自己的归档能力，二来或许也能帮助到正在学习的人。

该站文档源码托管在 github 上，网站部署在 readthedoc，你如有也有兴趣参与本站的建设可以到 github 上克隆该仓库，修改或添加新的文档提交后请求合并即可，仓库管理员会进行审核，审核通过就可以在该站上看到你的提交。至于内容审核是为了提升文档质量，审核包括内容、用词方式、文档风格等方面。总之是为了保证该站高质量、可信赖的形象。

Vikings 瞄准优秀的技术学习、知识原理的讲解。做一个纯粹的学习站，在中国来说程序员更熟悉 CSDN，CSDN 虽然被许多人鄙视但是它也为许多初学者提供了巨大帮助。自己前期也在 CSDN 上查找资料，但是后面自己发现 CSDN 内容良莠不齐，且没有质量保证，广告偏多影响用户体验。针对以上问题，我觉得 WiKi 的就和本站理念相似，WiKi 专注名词解释。而本站专注技术的讲解，对技术学习讲解达成共识，而不再让读者感觉学习困惑。

Vikings 指的是以前的北欧维京海盗。本站就是一艘知识的海盗船，知识就是黄金，跟海盗喜欢掠夺黄金的特点一拍即合，所以取名为 Vikings。本站的品味正如你所看到的这样，如果你觉得不错，也希望有共识的朋友参与到本站的建设。

有任何问题或者想加入 Vikings 组织请联系: `Caffreyfans@163.com`
